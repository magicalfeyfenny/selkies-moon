# Full character portrait masters

The seven `*_full.kra` documents are the canonical full-resolution portrait
masters. They are registered as source-only in the central Krita manifest, so
the normal runtime exporter validates them but does not create same-stem PNG
review copies in the repository.

The immutable original sketches and sprites in
`../original_character_references/` remain the design authority. Editing a KRA
must preserve the selected character's identity from those references; a prior
generated derivative does not become design authority merely because it is
convenient.

`PORTRAIT_BRIEFS.md` records the character-specific constraints used to select
and maintain these masters. Keep that tracked brief aligned with any
creator-approved design-authority change.
